# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it by creating a new **Issue**.
We appreciate your effort to responsibly disclose your findings, and we will make every effort to acknowledge your report and fix the issue.

### How to Report

1.  Navigate to the **Issues** tab in the repository.
2.  Click on **New Issue**.
3.  Provide a detailed description of the vulnerability.
4.  Include steps to reproduce the issue.

We will review the issue and work towards a resolution.
